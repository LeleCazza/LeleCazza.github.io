ESERCIZIO 1

Sito di riferimento per svolgere gli esercizi: W3schools

  1. Crea un documento di testo
    • Desktop - tasto destro del mouse - Nuovo - Documento di testo
  2. Salva il documento di testo come pagina html rinominandolo: esercizio1.html
    • File - Salva con nome... - Desktop - "esecizio1.html" - Salva
  3. Elimina il documento di testo inizialmente creato (mantieni solo la versione html appena creata)
  4. Apri il file "esercizio1.html" con il programma Blocco note (per modificarlo)
    • tasto destro del mouse sul file - Apri con - Blocco note
  5. Apri il file "esercizio1.html" con il programma Google Chrome (per visualizzarlo)
    • tasto destro del mouse sul file - Apri con - Google Chrome
  6. Dichiara all'interno del file che questo sarà un file HTML inserendo come prima riga l'istruzione: <!DOCTYPE html>
  7. ESERCIZIO 1: crea una pagina web statica con gli stessi elementi presenti nell'immagine di esempio allegata (immagine).
    • Titolo della pagina
    • Intestazione
    • Paragrafo
    • Lista di elementi
    • Immagine
Soluzione esercizio: soluzione.

ESERCIZIO 2

Sito di riferimento per svolgere gli esercizi: W3schools

Personalizza la pagina HTML allegata: tabella.html:

  • Rinomina il file esercizio_2_html_tabella.html nominandolo pagina1.html
  • Scegli un argomento e un tema a scelta
  • Utilizza solamente codice HTML, non usare il CSS (attributo o tag: style)
  • Inserisci elementi HTML personalizzati a scelta nelle varie sezioni della pagina
  • Riutilizza in almeno una sezione della pagina il tag <table>
Il testo e gli elementi presenti nel file allegato sono solamente a scopo indicativo, vanno modificati (non modificare la struttura base della tabella).

LA PAGINA WEB È STATA CREATA UTILIZZANDO I SEGUENTI TAG HTML:
  • STRUTTURA HTML DI UNA PAGINA WEB: <html>, <head>, <body>
  • DEFINIZIONE TITOLO DELLA PAGINA: <title>
  • DEFINIZIONE DI TABELLA: <table>
    • DEFINIZIONE DI RIGA: <tr>
    • DEFINIZIONE DI CELLA: <td>
  • DEFINIZIONE DI INTESTAZIONE: <h1>
  • DEFINIZIONE DI PARAGRAFO: <p>
  • DEFINIZIONE DI IMMAGINI: <img>
  • DEFINIZIONE DI I-FRAME: <iframe>
LA PAGINA WEB È STATA CREATA UTILIZZANDO I SEGUENTI ATTRIBUTI HTML:
  • ATTRIBUTI DEL TAG HTML <table>: width, height, border
  • ATTRIBUTI DEL TAG HTML <tr>: height, align, bgcolor
  • ATTRIBUTI DEL TAG HTML <td>: width, colspan
  • ATTRIBUTI DEL TAG HTML <img>: src, width
  • ATTRIBUTI DEL TAG HTML <iframe>: src, width

ESERCIZIO 3

Sito di riferimento per svolgere gli esercizi: W3schools

Prendi spunto dalla pagina HTML allegata: font&stile.html:

  • Crea una nuova pagina html nominandola: pagina2.html
  • Scegli un argomento e un tema a scelta
  • Utilizza solamente codice HTML, non usare il CSS (attributo o tag: style)
  • Inserisci elementi HTML personalizzati a scelta nelle varie sezioni della pagina
  • Riutilizza in almeno una sezione della pagina i seguenti tag:
    • <h1>
    • <p>
    • <font>
    • <b>
    • <i>
    • <u>
    • <a>
    • <br>
  • Riutilizza gli stessi attributi del tag <body> modificandone i valori a tuo piacimento
Il testo e gli elementi presenti nel file allegato sono solamente a scopo indicativo, vanno modificati (non modificare la struttura dell'head ma modificane gli elementi in modo opportuno).

LA PAGINA WEB È STATA CREATA UTILIZZANDO I SEGUENTI TAG HTML:
  • DEFINIZIONE DI METADATI: <meta>
  • DEFINIZIONE DI FONT: <font>
  • DEFINIZIONE DI CARATTERE IN GRASSETTO: <b>
  • DEFINIZIONE DI CARATTERE IN CORSIVO: <i>
  • DEFINIZIONE DI CARATTERE SOTTOLINEATO: <u>
  • DEFINIZIONE DI INTERRUZIONE DI RIGA: <br>
  • DEFINIZIONE DI LINK: <a>
LA PAGINA WEB È STATA CREATA UTILIZZANDO I SEGUENTI ATTRIBUTI HTML:
  • ATTRIBUTI DEL TAG HTML <html>: lang
  • ATTRIBUTI DEL TAG HTML <meta>: charset, name, content
  • ATTRIBUTI DEL TAG HTML <body>: leftmargin, topmargin, text, background, link, alink, vlink
  • ATTRIBUTI DEL TAG HTML <h1> E DEL TAG HTML <p>: align
  • ATTRIBUTI DEL TAG HTML <font>: size, face, color
  • ATTRIBUTI DEL TAG HTML <a>: href, target

ESERCIZIO 4

Sito di riferimento per svolgere gli esercizi: W3schools

Prendi spunto dalla pagina HTML allegata: div&form.html:

  • Crea una nuova pagina html nominandola: pagina3.html

  • ATTRIBUTI DEL TAG HTML <option>: value, selected

ESERCIZIO 5

Sito di riferimento per svolgere gli esercizi: W3schools

  1. Crea una cartella sul desktop nominandola "sito"
  2. All'interno della cartella appena creata crea altre due cartelle:
    • images
    • css
  3. Inserici le tre pagine HTML che hai realizzato negli esercizi precedenti: pagina1.html, pagina2.html, pagina3.html, all'interno della cartella "sito"
  4. Scarica la pagina: esercizio_5_index.html, rinominala in "index.html" e inseriscila all'interno della cartella "sito"
Struttura della cartella sito: struttura
Riferimenti relativi VS assoluti: W3schools

RIFERIMENTI RELATIVI
  • Scarica tre immagini a piacere e inseriscile all'interno della cartella "images"
  • Modifica la pagina "index.html" inserendo le immagini scaricate come ultimi elementi nel body (ricorda di utilizzare riferimenti relativi alle immagini).
RIFERIMENTI ASSOLUTI
  • Modifica la pagina "index.html" inserendo altre tre immagini direttamente dal web (ricorda di utilizzare riferimenti assoluti alle immagini)

Consegna una cartella compressa contenente il sito realizzato.
