Utilizzando Flowgorithm: download software
Una volta terminato l'esercizio consegna un file in formato .fprg (formato di flowgorithm) e un file in formato .png contenente la stampa del diagramma di flusso creato per risolvere ognuno dei seguenti esercizi:
CREARE FILE .fprg:

  1. Nel menu in alto di flowgorithm selezionare "File"
  2. Selezionare "Salva"
CREARE FILE .png:
  1. Nel menu in alto di flowgorithm selezionare "Strumenti"
  2. Selezionare "Esporta immagine in un file"
  3. Cliccare su "Crea"
