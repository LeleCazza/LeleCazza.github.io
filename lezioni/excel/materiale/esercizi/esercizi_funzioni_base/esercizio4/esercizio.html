Crea un foglio di calcolo per organizzare e tenere traccia del proprio utilizzo settimanale del telefono. Scegli le 5 applicazioni che utilizzi maggiormente e crea una tabella per visualizzare i seguenti dati (Ricordati di utilizzare i formati di cella opportuni):
  • Totale settimanale di utilizzo per ogni applicazione
  • Durata media settimanale di utilizzo per ogni applicazione
  • Durata massima settimanale di utilizzo per ogni applicazione
  • Durata minima settimanale di utilizzo per ogni applicazione
  • Totale giornaliero di utilizzo delle 5 applicazioni
  • Durata giornaliera media di utilizzo delle 5 applicazioni
  • Durata giornaliera massima di utilizzo delle 5 applicazioni
  • Durata giornaliera minima di utilizzo delle 5 applicazioni
  • Totale generale di utilizzo settimanale delle 5 applicazioni
Una volta terminato l'esercizio concentrati sul migliorare il layout (la leggibilità) della tabella appena creata.

Consegna il foglio di calcolo in due copie:
  1. Nel formato del foglio di calcolo utilizzato
  2. Nel formato .pdf (Portable Document Format)
OPERATORI BASE
  • Somma tra due celle: = cella1 + cella2
  • Sottrazione tra due celle: = cella1 - cella2
  • Moltiplicazione tra due celle: = cella1 * cella2
  • Divisione tra due celle: = cella1 / cella2
FUNZIONI BASE
  • Somma tra N celle: = SOMMA (cella1 : cellaN)
  • Media tra N celle: = MEDIA (cella1 : cellaN)
  • Massimo tra N celle: = MAX (cella1 : cellaN)
  • Minimo tra N celle: = MIN (cella1 : cellaN)
FORMATI BASE
  • VALUTA (€) esempio: 23,01€ (ventitré euro e un centesimo)
  • DURATA (hh.mm.ss) esempio: 2.30.00 (due ore, 30 minuti, 0 secondi)
