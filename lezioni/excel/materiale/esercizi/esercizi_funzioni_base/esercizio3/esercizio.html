Crea un foglio di calcolo per organizzare e tenere traccia delle seguenti informazioni: Un'azienda di videogiochi vuole calcolare i propri guadagni della settimana. Per ogni giorno della settimana viene fornito il guadagno giornaliero (a vostra scelta) per 10 diversi videogiochi (a vostra scelta). Calcola usando le funzioni e/o operazioni opportune:
  • Il guadagno totale settimanale per ogni singolo videogioco
  • Il guadagno totale generale aziendale
  • Il guadagno totale giornaliero aziendale
  • Il guadagno medio giornaliero aziendale
  • Il guadagno massimo giornaliero aziendale
  • Il guadagno minimo giornaliero aziendale
Una volta terminato l'esercizio concentrati sul migliorare il layout (la leggibilità) della tabella appena creata. Ricordati di utilizzare i formati di cella opportuni.

Consegna il foglio di calcolo in due copie:
  1. Nel formato del foglio di calcolo utilizzato
  2. Nel formato .pdf (Portable Document Format)
OPERATORI BASE
  • Somma tra due celle: = cella1 + cella2
  • Sottrazione tra due celle: = cella1 - cella2
  • Moltiplicazione tra due celle: = cella1 * cella2
  • Divisione tra due celle: = cella1 / cella2
FUNZIONI BASE
  • Somma tra N celle: = SOMMA (cella1 : cellaN)
  • Media tra N celle: = MEDIA (cella1 : cellaN)
  • Massimo tra N celle: = MAX (cella1 : cellaN)
  • Minimo tra N celle: = MIN (cella1 : cellaN)
FORMATI BASE
  • VALUTA (€) esempio: 23,01€ (ventitré euro e un centesimo)
