ESERCIZIO 1

Modifica il seguente foglio di calcolo. inserendo:

  • Alunni a scelta
  • Valutazioni a scelta
  • Medie delle valutazioni inserite
  • Bonus per ogni valutazione utilizzando RIFERIMENTI ASSOLUTI
  • Voto massimo per ogni disciplina
  • Voto minimo per ogni disciplina
Rircorda di non eseguire i calcoli con la calcolatrice o a mano ma di utilizzare le funzioni base, in particolare utilizza la funzione se per determinare se un voto è sufficiente o è insufficiente:
  • FUNZIONE SE
    • Nel caso la codizione sia verificata effettua i comandi presenti nella sezione vero, in alternativa effettua i comandi presenti nella sezione falso:
      =SE(condizione ; comandi_da_effettuare_se_vero ; comandi_da_effettuare_se_falso)
Utilizza la FORMATTAZIONE CONDIZIONALE per colorare le celle risultanti dalla funzione se:
  • Colora la cella di VERDE se il voto è sufficiente
  • Colora la cella di ROSSO se il voto è insufficiente
Consegna un file in formato .xlsx

ESERCIZIO 2

Utilizza la FORMATTAZIONE CONDIZIONALE, la FUNZIONE SE, i RIFERIMENTI ASSOLUTI e le:

  • FUNZIONI LOGICHE
    • Restituisce VERO se tutte le condizioni hanno valore VERO, FALSO altrimenti: =E(condizione1 ; condizione2 ; ecc...)
    • Restituisce VERO se una delle condizioni ha valore VERO, FALSO altrimenti: =O(condizione1 ; condizione2 ; ecc...)
    • Inverte in valore logico dell'argomento, restituisce FALSO per un argomento VERO e VERO per un argomento FALSO: =NON(argomento)
Per effettuare le seguenti operazioni sul foglio di calcolo allegato: foglio di calcolo
  1. Se la differenza reti (goal fatti - goal subiti) è maggiore di 5 inserire nella colonna ON FIRE la parola "SI", se no inserire la parola "NO".
  2. Se i gol fatti sono maggiori o uguali a 6 cololare la cella di verde, se no, colorare la cella di rosso.
  3. Se i gol subiti sono minori o uguali a 4 colorare la cella di verde, se no, cololare la cella di rosso.
  4. Se le vittorie sono maggiori o uguali a 3 e i goal fatti sono maggiori o uguali a 8 inserie una X nella colonna TOP.
Consegna un file in formato xlsx.

Il foglio di calcolo allegato contiene una macro excel, che modifica i valori contenuti nelle celle D3 : D12 con valori casuali tra 700 e 2000 euro alla pressione del tasto "SCARICA QUOTAZIONI". Completa l'esercizio e aggiorna le quotazioni quante volte vuoi, verificando di aver inserito le formattazioni, le funzioni e i riferimenti in modo corretto.
Per poter eseguire la macro presente nel foglio di calcolo:

  • Sbloccare la protezione inserita da excel di default (guida macro excel):
    1. Clicca col tasto destro del mouse sul file excel
    2. Seleziona dal menù la voce "Proprietà"
    3. Clicca sulla spunta "Annulla Blocco"
    4. Clicca sul bottone "Applica"
    5. Clicca sul bottone "OK"
  • Apri il file excel e clicca il pulsante "Abilita contenuto" presente nell'AVVISO DI SICUREZZA che notifica all'utente "Le macro sono state disabilitate"
