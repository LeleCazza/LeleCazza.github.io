1. Utilizza il software: diagrams.net per realizzare lo schema ER concettuale del contesto seguente. Consegna un file .drawio e un file .png contenente il diagramma ER;
  2. Utilizza un software di videoscrittura per convertire lo shema ER concettuale (vedi soluzione modello concettuale) nello schema logico corrispondente. Consegna un file di testo contenente lo schema logico;
  3. Utilizza l'ambiente di sviluppo: laragon.org per implementare il database del modello logico creato precedentemente. Consegna un file .sql contenente il database;
Si vuole progettare un database per una ditta di autotrasporti. Le informazioni di interesse riguardano gli automezzi , gli autisti e i viaggi. I viaggi ricoprono il territorio nazionale e hanno sempre una città di partenza, una città di arrivo e una breve descrizione del tipo di carico trasportato. Per i viaggi più lunghi sono previsti due autisti che si alternano alla guida.
Aggiungi possibili attributi alle entità, ogni entità dovrà possedere almeno tre attributi.
