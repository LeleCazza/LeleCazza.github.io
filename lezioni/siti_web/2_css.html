ESERCIZIO 1

Sito di riferimento per svolgere gli esercizi: W3schools

Utilizza TUTTE E TRE le modalità CSS:

  • modalità CSS esterno: viene utilizzata quando si vuole modificare lo stile di più pagine contemporaneamente.
  • modalità CSS interno: viene utilizzata quando si vuole modificare lo stile di una singola pagina aggiungendo molte proprietà CSS agli elementi HTML.
  • modalità CSS in linea: viene utilizzata quando si vuole modificare lo stile di un singolo e specifico elemento HTML aggiungendo poche proprietà CSS all'elemento.
per modificare il sito allegato (sito) in modo che:
  • Tutte le pagine del sito abbiano l'intestazione della pagina centrata orizzontalmente;
  • Ogni pagina abbia:
    • L'intestazione di un colore differente
    • Lo sfondo di un colore differente
  • Modifica la tabella e i link presenti nell'index.html personalizzando lo stile a tuo piacimento e inserendo tutte le proprietà CSS che ritieni necessarie

ESERCIZIO 2

Sito di riferimento per svolgere gli esercizi: W3schools

Utilizza TUTTI E TRE i principali selettori CSS:

  • selettore CSS tag: viene utilizzato quando si vuole modificare lo stile di tutti i tag specificati dal selettore.
  • selettore CSS classe: viene utilizzato quando si vuole modificare lo stile di un'insieme di elementi che devono avere tutti le stesse proprietà CSS.
  • selettore CSS identificativo: viene utilizzato quando si vuole modificare lo stile di un singolo e specifico elemento HTML aggiungendo specifiche proprietà CSS all'elemento.
per modificare il sito allegato (sito) in modo che:
  • La PRIMA PAGINA del sito abbia tre sezioni come nell'immagine allegata. Per ottenere il risultato desiderato effettua i seguenti passi:
    • Inserisci un <div> contenitore delle tre sezioni
    • Inserisci un <div> nel contenitore per ogni sezione
    • Inserisci un <h2> con il titolo della sezione per ogni sezione
    • Assegna i selettori corretti (classi e/o identificativi)
    • Modifica il file css esterno aggiungendo le seguenti proprietà (esempio):
      • CONTENITORE: bordo solido di tre pixel di colore rosso allineato al centro e con grandezza 100%
      • SEZIONE: bordo solido di un pixel di colore nero e con grandezza 100%
  • La SECONDA PAGINA del sito abbia tre sezioni con CSS libero
  • La TERZA PAGINA del sito abbia tre sezioni con CSS libero
