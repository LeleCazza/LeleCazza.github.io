Sito di riferimento per svolgere gli esercizi: W3schools

    • tasto destro del mouse sul file - Apri con - Blocco note

    • tasto destro del mouse sul file - Apri con - Google Chrome

    • Titolo della pagina
    • Intestazione
    • Paragrafo

Sito di riferimento per svolgere gli esercizi: W3schools

Personalizza la pagina HTML allegata: tabella.html:

  • Rinomina il file esercizio_HTML_2.html nominandolo pagina1.html
  • Scegli un argomento e un tema a scelta
  • Utilizza solamente codice HTML, non usare il CSS (attributo o tag: style)
  • Inserisci elementi HTML personalizzati a scelta nelle varie sezioni della pagina
  • Riutilizza in almeno una sezione della pagina il tag <table>
Il testo e gli elementi presenti nel file allegato sono solamente a scopo indicativo, vanno modificati (non modificare la struttura base della tabella).

LA PAGINA WEB È STATA CREATA UTILIZZANDO I SEGUENTI TAG HTML:
  • STRUTTURA HTML DI UNA PAGINA WEB: <html>, <head>, <body>
  • DEFINIZIONE TITOLO DELLA PAGINA: <title>
  • DEFINIZIONE DI TABELLA: <table>
    • DEFINIZIONE DI RIGA: <tr>
    • DEFINIZIONE DI CELLA: <td>
  • DEFINIZIONE DI INTESTAZIONE: <h1>
  • DEFINIZIONE DI PARAGRAFO: <p>
  • DEFINIZIONE DI IMMAGINI: <img>
  • DEFINIZIONE DI I-FRAME: <iframe>
LA PAGINA WEB È STATA CREATA UTILIZZANDO I SEGUENTI ATTRIBUTI HTML:
  • ATTRIBUTI DEL TAG HTML <table>: width, height, border
  • ATTRIBUTI DEL TAG HTML <tr>: height, align, bgcolor
  • ATTRIBUTI DEL TAG HTML <td>: width, colspan
  • ATTRIBUTI DEL TAG HTML <img>: src, width
  • ATTRIBUTI DEL TAG HTML <iframe>: src, width

ESERCIZIO 4

Sito di riferimento per svolgere gli esercizi: W3schools

Modifica le DUE PAGINE HTML: tabella1.html e tabella2.html create in precedenza per ottenere due differenti siti web:

NON UTILIZZARE L'ATTRIBUTO O IL TAG: STYLE

  • tabella1.html: sito web di un'azienda di scarpe a tua scelta. Il nome dell'azienda scelta con il proprio logo allineato alla sua destra/sinistra deve trovarsi nell'header della pagina (sezione in alto). La pagina consiste nell'esposizione di tre scarpe, una per ogni riga, dove: a sinistra e a destra devono trovarsi le immagini del modello di scarpa descritto nella colonna centrale tramite il nome del modello e una breve descrizione. Nel footer della pagina (sezione in basso) devono essere inserite le informazione di contatto dell'azienda scelta (numero di telefono, sede, partita IVA, mail, ecc...). RICORDA DI INSERIRE I TITOLI ALL' INTERNO DEL TAG <h1> E I TESTI ALL'INTERNO DEL TAG <p>. Per allineare i contenuti puoi utilizzare l'attributo della riga align="center".
    ESEMPIO: tabella1
  • tabella2.html: sito web di un'azienda a scelta. L'impaginazione della pagina è libera. Ricorda che nell'header devono essere presenti il nome dell'azienda e il logo, nel footer le informazioni di contatto.
Consegna i due file HTML modificati

  • Crea una nuova pagina html nominandola: pagina2.html
  • Scegli un argomento e un tema a scelta
  • Utilizza solamente codice HTML, non usare il CSS (attributo o tag: style)
  • Inserisci elementi HTML personalizzati a scelta nelle varie sezioni della pagina
  • Riutilizza in almeno una sezione della pagina i seguenti tag:
    • <h1>
    • <p>
    • <font>
    • <b>
    • <i>
    • <u>
    • <a>
    • <br>

Il testo e gli elementi presenti nel file allegato sono solamente a scopo indicativo, vanno modificati (non modificare la struttura dell'head ma modificane gli elementi in modo opportuno).

  • DEFINIZIONE DI FONT: <font>
  • DEFINIZIONE DI CARATTERE IN GRASSETTO: <b>
  • DEFINIZIONE DI CARATTERE IN CORSIVO: <i>
  • DEFINIZIONE DI CARATTERE SOTTOLINEATO: <u>
  • DEFINIZIONE DI INTERRUZIONE DI RIGA: <br>
  • DEFINIZIONE DI LINK: <a>
LA PAGINA WEB È STATA CREATA UTILIZZANDO I SEGUENTI ATTRIBUTI HTML:
  • ATTRIBUTI DEL TAG HTML <html>: lang
  • ATTRIBUTI DEL TAG HTML <meta>: charset, name, content
  • ATTRIBUTI DEL TAG HTML <body>: leftmargin, topmargin, text, background, link, alink, vlink
  • ATTRIBUTI DEL TAG HTML <h1> E DEL TAG HTML <p>: align
  • ATTRIBUTI DEL TAG HTML <font>: size, face, color
  • ATTRIBUTI DEL TAG HTML <a>: href, target
