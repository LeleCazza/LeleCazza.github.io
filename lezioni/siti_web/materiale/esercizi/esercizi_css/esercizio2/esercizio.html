Utilizza TUTTI E TRE i principali selettori CSS:
  • selettore CSS tag: viene utilizzato quando si vuole modificare lo stile di tutti i tag specificati dal selettore.
  • selettore CSS classe: viene utilizzato quando si vuole modificare lo stile di un'insieme di elementi che devono avere tutti le stesse proprietà CSS.
  • selettore CSS identificativo: viene utilizzato quando si vuole modificare lo stile di un singolo e specifico elemento HTML aggiungendo specifiche proprietà CSS all'elemento.
