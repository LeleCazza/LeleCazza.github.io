Utilizza TUTTE E TRE le modalità CSS:
  • modalità CSS esterno: viene utilizzata quando si vuole modificare lo stile di più pagine contemporaneamente.
  • modalità CSS interno: viene utilizzata quando si vuole modificare lo stile di una singola pagina aggiungendo molte proprietà CSS agli elementi HTML.
  • modalità CSS in linea: viene utilizzata quando si vuole modificare lo stile di un singolo e specifico elemento HTML aggiungendo poche proprietà CSS all'elemento.
