Prendi spunto dalla pagina HTML allegata: div&form.html:
  • Crea una nuova pagina html nominandola: pagina3.html
  • Scegli un argomento e un tema a scelta
  • Utilizza solamente codice HTML, non usare il CSS (attributo o tag: style)
  • Inserisci elementi HTML personalizzati a scelta nelle varie sezioni della pagina
  • Crea un form di registrazione per la tua pagina web:
    • Riutilizzando e personalizzando i tag presenti nella pagina allegata
    • Aggiungendo almeno un nuovo tag di input (non presente nella pagina allegata)
Il testo e gli elementi presenti nel file allegato sono solamente a scopo indicativo, vanno modificati (non modificare la struttura dell'head ma modificane gli elementi in modo opportuno).

LA PAGINA WEB È STATA CREATA UTILIZZANDO I SEGUENTI TAG HTML:
  • DEFINIZIONE DI SEZIONE: <div>
  • DEFINIZIONE DI FORM: <form>
  • DEFINIZIONE DI AREA DATI: <fieldset>
  • DEFINIZIONE DI LEGENDA AREA DATI: <legend>
  • DEFINIZIONE DI ETICHETTA DATO: <label>
  • DEFINIZIONE DI DATO DI INPUT: <input>
  • DEFINIZIONE DI SELETTORE: <select>
  • DEFINIZIONE DI OPZIONE IN UN SELETTORE: <option>
  • DEFINIZIONE DI AREA DI TESTO: <textarea>
LA PAGINA WEB È STATA CREATA UTILIZZANDO I SEGUENTI ATTRIBUTI HTML:
  • ATTRIBUTI DEL TAG HTML <div>: align
  • ATTRIBUTI DEL TAG HTML <form>: action
  • ATTRIBUTI DEL TAG HTML <input>: type, name, value
  • ATTRIBUTI DEL TAG HTML <select>: name
  • ATTRIBUTI DEL TAG HTML <option>: value, selected
  • ATTRIBUTI DEL TAG HTML <textarea>: name, rows, cols
