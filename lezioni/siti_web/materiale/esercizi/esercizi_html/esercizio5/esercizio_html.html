HTML Quiz

You can test your HTML skills with W3Schools' Quiz.

The test contains 40 questions and there is no time limit. The test is not official, it's just a nice way to see how much you know, or don't know, about HTML. You will get 1 point for each correct answer. At the end of the Quiz, your total score will be displayed. Maximum score is 40 points.

HTML Exercises

Test your HTML skills with exercises from all categories.

The exercises are a mix of "multiple choice" and "fill in the blanks" questions. There are between 3 and 9 questions in each category. The answer can be found in the corresponding tutorial chapter. If you're stuck, or answer wrong, you can try again or hit the "Show Answer" button to see the correct answer.

HTML Code Challenges

Test your HTML skills with code challenges from all categories.

Code challenges are interactive coding exercises where you write HTML code to achieve a specific result. Each challenge presents you with a goal and lets you practice writing real HTML code.
