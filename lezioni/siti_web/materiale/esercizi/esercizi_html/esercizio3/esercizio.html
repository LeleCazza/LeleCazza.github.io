Crea DUE PAGINE HTML: tabella1.html e tabella2.html per ottenere un'impaginazione identica a quella presente nelle due immagini allegate:
TABELLA 1: 5 RIGHE, 3 COLONNE
TABELLA 2: 3 RIGHE, 4 COLONNE

CREA LE PAGINE WEB UTILIZZANDO I SEGUENTI TAG HTML:
  • STRUTTURA HTML DI UNA PAGINA WEB: <html>, <head>, <body>
  • DEFINIZIONE TITOLO DELLA PAGINA: <title>
  • DEFINIZIONE DI TABELLA: <table>
    • DEFINIZIONE DI RIGA: <tr>
    • DEFINIZIONE DI CELLA: <td>
CREA LE PAGINE WEB UTILIZZANDO I SEGUENTI ATTRIBUTI HTML:
  • ATTRIBUTI DEL TAG HTML <table>: width, height, border
  • ATTRIBUTI DEL TAG HTML <tr>: height, align
  • ATTRIBUTI DEL TAG HTML <td>: width, colspan
